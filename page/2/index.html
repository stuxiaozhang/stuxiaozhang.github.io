一维前缀和（计算区域内的和 ）

思路

  • 作用：能快速求出原数组中一段区间的数字和。

  • 计算公式：计算 \([l, r]\) 区间和前缀和 S[r] - S[l - 1]

  • a[l] + ... + a[r] = S[r] - S[l - 1]

    如何推导？ \[ S_r = a_1 + a_2 + .... + a_{l-1} + a_l + ... + a_r \]

    \[ S_{l-1} = a_1 + a_2 + .... + a_{l-1} \]

    \((1) - (2)\) 得： \[ S_r - S_{l-1} = a_l + .... + a_r \]

  • 为什么下标要从 \(1\) 开始？
