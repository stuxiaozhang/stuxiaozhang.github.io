TransD：Knowledge Graph Embedding via Dynamic Mapping Matrix

本文最后更新于：2021年9月12日

TransD：Knowledge Graph Embedding via Dynamic Mapping Matrix

本文是自动化所赵军、刘康老师团队发表在 ACL 2015 上的工作

Motivation

TransR 存在一些问题：

  1. 对于特定的关系 \(r\)，所有的实体都共享相同的映射矩阵 \(M_r\)。然而，由关系链接的实体总是包含各种类型和属性。

  2. 投影是实体和关系之间的交互过程，映射矩阵只能由关系决定是不合理的。

  3. 矩阵向量乘法使其具有大量计算，并且当关系数大时，它还具有比 TransE 和 TransH 更多的参数。 由于复杂性，TransR / CTransR 难以应用于大规模知识图。

Basic Idea

D 代表 Dynamic，是针对 TransR 参数量太大做的改进。主要思想是：实体和关系共同构建映射矩阵。具体地，每个实体和关系由两个向量（meaning vector 和 projection vector）表示，一个表示本身的 embedding，另一个用于构建映射矩阵。每个“实体-关系”对使用的映射矩阵是不同的，头尾实体单独投影。

TransD

在 TransD 中，作者为每个实体和关系定义了两个向量。第一个向量表示实体或关系的含义，另一个向量（称为投影向量）表示如何将实体嵌入关系向量空间，并将其用于构造映射矩阵。

\(\mathbf{M}_{r h}, \mathbf{M}_{r t}\) 分别是实体 \(h , t\) 的映射矩阵，\(\mathbf{h}_{ip},\mathbf{t}_{ip}\) 及关系 \(\mathbf{r}_{p}\) 为投影向量，\(\mathbf{h}_{i\perp},\mathbf{t}_{i\perp}\) 分别为头尾实体的投影向量。

对于一个三元组\((h,r,t)\)，构建映射矩阵： \[ \begin{aligned} \mathbf{M}_{r h} &=\mathbf{r}_{p} \mathbf{h}_{p}^{\top}+\mathbf{I}^{m \times n} \\ \mathbf{M}_{r t} &=\mathbf{r}_{p} \mathbf{t}_{p}^{\top}+\mathbf{I}^{m \times n} \end{aligned} \] 映射矩阵由实体和关系决定，这种操作使得两个投影向量充分交互。

投影向量： \[ \mathbf{h}_{\perp}=\mathbf{M}_{r h} \mathbf{h}, \quad \mathbf{t}_{\perp}=\mathbf{M}_{r t} \mathbf{t} \] 三元组得分函数： \[ f_{r}(\mathbf{h}, \mathbf{t})=-\left\|\mathbf{h}_{\perp}+\mathbf{r}-\mathbf{t}_{\perp}\right\|_{2}^{2} \] 归一化约束： \[ \|\mathbf{h}\|_{2} \leq 1,\|\mathbf{t}\|_{2} \leq 1,\|\mathbf{r}\|_{2} \leq 1,\left\|\mathbf{h}_{\perp}\right\|_{2} \leq 1 \text { and }\left\|\mathbf{t}_{\perp}\right\|_{2} \leq1 \] 目标函数： \[ L=\sum_{\xi \in \Delta} \sum_{\xi^{\prime} \in \Delta^{\prime}}\left[\gamma+f_{r}\left(\xi^{\prime}\right)-f_{r}(\xi)\right]_{+} \]

与 TransE、TransH 和 TransR 的关系

  1. 与 TransE 的关系

    当实体、关系维度相同 且 所有的投影向量设为 0 时，退化为 TransE

  2. 与 TransH 的关系

    当实体、关系维度相同时，TransD 的投影操作可展开为： \[ \begin{aligned} \mathbf{h}_{\perp} &=\mathbf{M}_{r h} \mathbf{h}=\mathbf{h}+\mathbf{h}_{p}^{\top} \mathbf{h} \mathbf{r}_{p} \\ \mathbf{t}_{\perp} &=\mathbf{M}_{r t} \mathbf{t}=\mathbf{t}+\mathbf{t}_{p}^{\top} \mathbf{t r}_{p} \end{aligned} \] 而 TransH 的投影操作为： \[ \begin{aligned} \mathbf{h}_{\perp}=\mathbf{h}-\mathbf{w}_{r}^{\top} \mathbf{h} \mathbf{w}_{r} \\ \mathbf{t}_{\perp}=\mathbf{t}-\mathbf{w}_{r}^{\top} \mathbf{t} \mathbf{w}_{r} \end{aligned} \] 由此可看到，TransH 中投影的方向只取决于 \(r\), 而在 TransD 中, 由 \(e\) 和 \(r\) 共同决定.

  3. 与 TransR 的关系

    TransR 为每个关系定义一个投影矩阵 \(M_r\)。而 TransD 通过为每个实体和关系设置投影向量，为每个三元组动态构建两个映射矩阵： \[ \begin{aligned} \mathbf{M}_{r h} &=\mathbf{r}_{p} \mathbf{h}_{p}^{\top}+\mathbf{I}^{m \times n} \\ \mathbf{M}_{r t} &=\mathbf{r}_{p} \mathbf{t}_{p}^{\top}+\mathbf{I}^{m \times n} \end{aligned} \] 投影向量为： \[ \begin{aligned} \mathbf{h}_{\perp} &=\mathbf{M}_{r h} \mathbf{h}=\mathbf{h}_{p}^{\top} \mathbf{h} \mathbf{r}_{p}+\left[\mathbf{h}^{\top}, \mathbf{0}^{\top}\right]^{\top} \\ \mathbf{t}_{\perp} &=\mathbf{M}_{r t} \mathbf{t}=\mathbf{t}_{p}^{\top} \mathbf{t r}_{p}+\left[\mathbf{t}^{\top}, \mathbf{0}^{\top}\right]^{\top} \end{aligned} \] 没有“矩阵-向量”相乘的操作，大大减少了参数量和计算量。据原文称，TransD 比 TransR 快约三倍。

Experiments

Triplets Classification

  1. 在WN11上，Trans 胜过包括 TransE、TransH和 TransR/CTransR 在内的所有其他先前模型，尤其是NTN（E）
  2. 在FB13上，TransD 的分类精度显著高于 TransE、TransH 和 TransR/CTransR，接近NTN（E）
  3. 在大多数情况下，“bern” 抽样法比 “unif” 抽样法效果更好。

Summary

  1. 作者构建了一个新颖的模型 TransD，通过同时考虑实体和关系的多样性，为每一个实体-关系构建动态映射矩阵。它为实体表示映射到关系向量空间提供灵活的样式。
  2. 与 TransR / CTransR 相比，TransD 具有更少的参数并且没有矩阵向量乘法
  3. 在实验中，作者的方法优于之前的模型。
