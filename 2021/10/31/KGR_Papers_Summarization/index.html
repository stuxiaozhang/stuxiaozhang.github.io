1. Go for a Walk and Arrive at the Answer: Reasoning Over Paths in Knowledge Bases using Reinforcement Learning

  • 作者：Rajarshi Das, Shehzaad Dhuliawala，ICLR 2018，paper code
  • Idea：本文提出了 MINERVA 模型，使用了 Reinforcement Learning 的方法，根据 query 在图谱上游走获得 path，抵达目标节点后停止，该方法可以通过在知识图谱上游走推理 path 获得答案，不需要对所有实体进行枚举。

2.【Multi-Hop KG】Multi-Hop Knowledge Graph Reasoning with Reward Shaping

  • Author：Xi Victoria Lin, Richard Socher, Caiming Xiong，EMNLP 2018，paper code

  • Idea：该论文基于上篇 MINERVA 指出了强化学习在 incomplete knowledge graph 中进行 reasoning 时存在的两个问题，

    1. 虚假负 reward：查询出的路径由于知识图谱中的信息缺失导致判定为错误路径
    2. 虚假路径：由假路径导致的偶然正确答案。由于强化学习是一种基于策略的算法，鼓励过去的行动，并反馈回较高的reward，因此会更偏向初期存在的虚假正确的情况。(false positive path)

    针对以上问题, 本文提出解决方案：Reward Shaping 和 Action dropout：

    1. Reward Shaping：通过采用 pretrained one-hop embedding model，对任意三元组有一个预期奖励，有效避免训练时 False Negatives 对测试效果造成的影响。
    2. Action dropout：对每一个路径上的中间实体，随机对其出边进行dropout，以强制有效探索不同的路径集，并稀释虚假路径的负面影响。

3.【CPL】Collaborative Policy Learning for Open Knowledge Graph Reasoning.

  • Author：Cong Fu, Tong Chen, Meng Qu, Woojeong Jin, Xiang Ren. EMNLP 2019. paper code

  • Idea：作者在“开放世界”环境下研究知识图推理，从背景语料库中抽取的新事实可用于促进路径发现。然后，依据此提出了一种新的协作策略学习（CPL）框架，以相互增强的方式联合训练两个 RL 代理。在 CPL 中，除了训练用于路径发现的 Reasoner Agent 之外，还引入了 Fact Extractor Agent，该代理根据推理过程和语料库的上下文学习从语料库中抽取相关事实的策略。在推理时，事实抽取代理仅使用信息量最大的边动态扩充图，从而使推理代理能够有效地识别正路径。

4.【DacKGR】Dynamic Anticipation and Completion for Multi-Hop Reasoning over Sparse Knowledge Graph

  • Author：Xin Lv, Xu Han, Lei Hou, Juanzi Li, Zhiyuan Liu.... EMNLP 2020. paper code

  • Idea：DacKGR 针对稀疏知识图谱做多跳推理。可以做多跳推理的模型主要有两类：KGE 模型和专门的多跳推理模型。但 KGE 做补全是计算出来的结果，不能给出推理路径之类的解释。而专门的多跳推理模型作用于稀疏知识图谱时面临两个问题：稀疏 KG 包含的信息少，路径选择困难；路径的缺乏使得预测目标实体变得困难。

    因此，为了解决在稀疏路径的 KG 进行多跳推理比较困难的问题，文章提出了 DacKGR，包括动态预测（Dynamic Anticipation）和动态补全（Dynamic Completion）两部分：

    1. Dynamic Anticipation：应用 KGE 模型预测尾实体信息用于 agent 选择正确的 action
    2. Dynamic Completion：在路径搜索过程中，动态地向 KG 中添加额外的边（关系），缓解 KG 的稀疏性问题。

1.【BIMR】Is Multi-Hop Reasoning Really Explainable? Towards Benchmarking Reasoning Interpretability

  • Author：Xin Lv, Yixin Cao, Lei Hou, Juanzi Li, Zhiyuan Liu.... EMNLP 2021

  • Idea：针对多跳推理没有量化的评价指标的问题，提出了衡量多跳路径可解释性得分的 BIMR，包括三个指标：PR、LI 和 GI，弥补了多跳推理效果没有量化评价指标的空白。

    作者说未来的工作可以研究：如何更好地将 挖掘出的规则 融入到 多跳推理模型 中，以获得更好的性能和可解释性。以后可以看一下 Rule mining 的 KGC.
