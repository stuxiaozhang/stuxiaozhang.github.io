DOM

DOM 介绍

DOM：将文档（页面）表现为结构化的表示方法，使每一个页面元素都是可操控，DOM将网页和脚本以及其他的编程语言联系了起来。通过 HTML DOM，可访问 JavaScript HTML 文档的所有元素。不同于 jQuery，DOM 是 JS 原生的， 可直接使用。

HTML DOM （文档对象模型）

当网页被加载时，浏览器会创建页面的文档对象模型（Document Object Model）。

HTML DOM 模型被构造为对象的树。

  • 特点：利用 js 控制页面中的所有元素，使页面更加“聪明”。

DOM 节点

在 HTML DOM 中，所有事物都是节点。DOM 是被视为节点树的 HTML 。

  • 整个文档是一个文档节点
  • 每个元素是元素节点
  • 元素内的文本是文本节点
  • 每个元素的属性是属性节点
  • 注释是注释节点

节点父、子和同胞

节点树中的节点彼此拥有层级关系。

父（parent）、子（child）和同胞（sibling）等术语用于描 述这些关系。父节点拥有子节点。同级的子节点被称为同胞。 在节点树中，顶端节点被称为根（root ）

  • 每个节点都有父节点、除了根（它没有父节点）
  • 一个节点可拥有任意数量的子
  • 同胞是拥有相同父节点的节点

  • <head> 元素是 <html> 元素的首个子节点
  • <body> 元素是 <html> 元素的最后一个子节点
  • <h1> 元素是 <body> 元素的首个子节点
  • <p> 元素是 <body> 元素的最后一个子节点

HTML DOM Document 对象

每个载入浏览器的 HTML 文档都会成为 Document 对象。

Document 对象使我们可以从脚本中对 HTML 页面中的所有元素进行访问。

提示：Document 对象是 Window 对象的一部分，可通过 window.document 属性对其进行访问。

响应事件

事件：事件是指被程序发现的行为或发生的事情，而且它可能会被程序处理

特点：js 的事件，都是以 on 开头，有 onclick、onload、onchange...

分类：表单事件、鼠标事件、键盘事件...

onload 和 onunload 事件

当用户进入或离开页面时，会触发 onload 和 onunload 事件。

onload 事件可用于检查访客的浏览器类型和版本，以便基于这些信息来加载不同版本的网页。

onload 和 onunload 事件可用于处理 cookies。

例：<body onload="checkCookies()">

onchange 事件

onchange 事件常用于输入字段的验证。

例：当用户改变输入字段的内容时，将调用 changeColor () 函数。

onmouseover 和 onmouseout 事件

onmouseover 和 onmouseout 事件可用于在鼠标指针移动到或离开元素时触发函数。
