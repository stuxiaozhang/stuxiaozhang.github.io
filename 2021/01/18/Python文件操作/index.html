Python文件操作

发表于 2021-01-18 分类于 编程语言 ， Python 阅读次数： Valine：

这篇文章整理了Python相关的文件操作：文件的读写，序列化等等。

1. 文件的创建与写入

1.1 open 函数获取文件对象

内置函数 open 函数：生成文件对象，进行创建、读写操作。

语法：open(path, mode) ，返回一个文件对象

  • path ：文件路径
  • mode ：操作模式

1.2 文件的创建与写入

常见的写入模式

模式可做操作介绍是否覆盖
w只能写若没有这个文件，则创建文件。若有这个文件，则覆盖文件中的内容。是
w+可读可写打开一个文件用于读写。是
wb可读可写二进制形式。若没有这个文件，则创建文件。若有这个文件，则覆盖文件中的内容。是
wb+可读可写二进制形式创建或追加内容（若无则创建，若有则继续以比特类型写入内容）。是
a可读可写在已存在的文件中追加内容。否，追加写
a+可读可写同上。多了可以读取的功能。否，追加写
ab+可读可写二进制形式追加读取内容。否，追加写

1.3 文件对象的写入保存

常见的文件对象操作方法

方法名参数介绍举例
writemessage写入信息f.write(‘hello\n’)
writelinesmessage_list批量写入f.writelines(['hello\n', 'world\n'])
close无关闭并保存文件。操作完必须用close方法！f.close()

1.5 文件对象的读取

常用的文件对象的读取

方法名参数介绍举例
read无返回整个文件字符串f.read()
readlines无返回文件列表f.readlines()
readline无返回文件中的一行f.readline()
mode无文件模式f.mode
name无返回文件名称f.name
closed无文件是否关闭f.closed

2. 序列化

序列化就是：将对象信息或者数据结构信息通过一定规则进行转换进行文件存储或者网络传输。

可序列化的数据类型：

  • number
  • str
  • list
  • tuple
  • dict（最常用）
